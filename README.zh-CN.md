## Usage

### Back-End

```bash
$ cd rdm-web/
$ mvn clean install -U
$ cd rdm-web/target 
$ java -jar rdm-web-0.0.1-SNAPSHOT.jar
```

### FRONT-End

```bash
$ cd rdm-web/web
$ npm install
$ npm start         # visit http://localhost:8000
```
